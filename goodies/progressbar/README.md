# Progress Bar

Simple green progress bar that fills the entire screen, useful for multistage tasks or showing that something is being performed.

## How to use:
**First way:** Copy over '_autorun.ds_' and all the PNG files on the SD card. When you plug in the device, the animation will run automatically, _everytime_.

**Second way:** Copy over '_progressbar.ds_' and all the PNG files on the SD card. When you plug in the device, the animation has to be started by executing '_progressbar.ds_' in its web interface.

**Note:** This example is purely visual, it doesnt represent _any_ progress of anything happening.
