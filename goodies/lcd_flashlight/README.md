# LCD Flashlight
Simple script that shows a white PNG on the screen that acts a fairly bright flashlight.

## How to set up:
1. Copy the 'flashlight.ds' and 'flashlight.png' files to the root of your SD card.

## How to use:
1. Plug in your device.
2. Execute the 'flashlight.ds' script via the Web UI or using the [DuckyScript Executer](goodies/duckyscript_executer_ui/README.md)
3. To stop it, press the button to reset your device.
